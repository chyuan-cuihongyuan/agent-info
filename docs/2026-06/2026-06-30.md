# 2026-06-30 AI 资讯

| 研究方向 | 大概讲了什么 | 文章链接 |
|---------|------------|---------|
| RLVR/训练方法 | 提出 Tandem RL，让强模型与冻结弱模型交替共同生成推理链，使 RLVR 在保持推理能力的同时生成更易被弱模型与人类理解的思维链 | [Tandem Reinforcement Learning with Verifiable Rewards](https://arxiv.org/abs/2606.28166) |
| Agent 框架/安全 | 提出 Agent 原生免疫系统的架构与分类法，为智能体系统设计可工程化的检测与防御机制 | [Agent-Native Immune System: Architecture, Taxonomy, and Engineering](https://arxiv.org/abs/2606.28270) |

| 多 Agent | 研究人格提示对多 Agent LLM 团队任务表现的影响，发现人格效应高度依赖任务结构：编码任务受影响小，开放协作与谈判受影响大 | [When Does Personality Composition Matter for Multi-Agent LLM Teams?](https://arxiv.org/abs/2606.27443) |
| 多 Agent/Agent 框架 | QueenBee Planner 引入技能演化的通信拓扑，降低 LLM 多智能体系统的 token 开销同时保持协作质量 | [QueenBee Planner: Skill-Evolving Communication Topologies for Token-Efficient LLM Multi-Agent Systems](https://arxiv.org/abs/2606.27492) |
| 代码生成 Agent | Glite ARF 用验证器驱动并行 LLM 编码 Agent 协作完成科研任务，提升代码 Agent 的研究产出效率 | [Glite ARF: Verifier-Driven Research with Parallel LLM Coding Agents](https://arxiv.org/abs/2606.27416) |
| 多 Agent | 研究延迟验证对多 Agent LLM 信念稳定性的影响，给出不稳定阈值与最优纠错器放置位置 | [Delayed Verification Destabilizes Multi-Agent LLM Belief](https://arxiv.org/abs/2606.27409) |
| 评估基准/Agent | SidConArena 构建开放式正和谈判博弈环境，用于评估 Agent 的协商与讨价还价能力 | [SidConArena: An Environment Evaluating Agents in Open-Ended, Positive-Sum Bargaining Game](https://arxiv.org/abs/2606.27397) |
| Agent 框架 | GenWorld 提供经验驱动的城市仿真基础设施，支撑大规模 LLM-Agent 研究的可扩展部署 | [GenWorld: Empirically Grounded Urban Simulation Infrastructure for Scalable LLM-Agent Studies](https://arxiv.org/abs/2606.27650) |
| 多 Agent | GBC 提出基于梯度的连接方法优化多智能体系统，使梯度可在 Agent 间传递以联合优化整体性能 | [GBC: Gradient-Based Connections for Optimizing Multi-Agent Systems](https://arxiv.org/abs/2606.28187) |
| LLM 推理 | 在狼人杀中引入小丑角色，测试 LLM 的多跳心理理论（Theory of Mind）推理能力 | [Triadic Werewolf: A Jester Role for Multi-Hop Theory of Mind in LLMs](https://arxiv.org/abs/2606.27909) |
| Agent/世界模型 | 研究图世界模型的长程 rollout 误差传播，区分拓扑放大与模型放大，提出 Error-Aware GWM 防止长程发散 | [Understanding Rollout Error in Graph World Models](https://arxiv.org/abs/2606.27780) |
| 多模态/推理 | MER-R1 用强化学习统一快思考的召回与慢思考的精度，通过双目标解耦与置信度校准让推理真正提升多模态情感识别 | [MER-R1: Multimodal Emotion Reasoning via Slow-Fast Thinking Synergy](https://arxiv.org/abs/2606.27652) |
| LLM 推理 | SD-GPS 将符号求解器作为执行 oracle 贯穿形式化与推理，结合可验证奖励的 RL 与定理提出 agent 提升几何推理 | [Verifiable Geometry Problem Solving: Solver-Driven Autoformalization and Theorem Proposing](https://arxiv.org/abs/2606.27926) |
| LLM 应用 | 京东 Oxygen AIIC 构建工业级 LLM/VLM 商品知识生产平台，覆盖数万品类、日处理数亿更新，搜索流量覆盖达 80.4% | [JD Oxygen AI Item Center (Oxygen AIIC)](https://arxiv.org/abs/2606.28070) |
| 多 Agent/对齐 | 在完全委托的 AI 协作体中研究价值约束下的信用分配，为多智能体协作的价值对齐提供机制 | [Towards Value-Constrained Credit Assignment in Fully Delegated AI Cooperatives](https://arxiv.org/abs/2606.28217) |
