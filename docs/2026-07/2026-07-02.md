# 2026-07-02 AI 资讯

| 研究方向 | 大概讲了什么 | 文章链接 |
|---------|------------|---------|
| 多模态/推理 | Hugging Face 与 Cerebras 合作构建开源级联语音到语音管线（Parakeet ASR + Gemma 4 VLM 在 Cerebras 上推理 + Qwen3TTS），借助高速推理实现实时自然语音对话 | [Hugging Face and Cerebras bring Gemma 4 to real-time voice AI](https://huggingface.co/blog/cerebras-gemma4-voice-ai) |
| 训练方法/可解释性 | 研究训练语言模型生成预测解释何时产生忠实"内省"而非表面模仿，发现基于固定反事实解释的训练能持续追踪行为漂移，为内省提供可泛化的后训练信号 | [Introspective Coupling: Self-Explanation Training Tracks Behavioral Change Despite Fixed Supervision](https://arxiv.org/abs/2606.32038) |
| RLHF/对齐 | 提出 RLMF 范式，依据模型自判断质量在偏好优化中重排候选，使 LLM 更忠实地表达内在不确定性，在忠实校准任务上超越标准 RL 最高 63% | [Reinforcement Learning with Metacognitive Feedback Elicits Faithful Uncertainty Expression in LLMs](https://arxiv.org/abs/2606.32032) |
| LLM 评估 | 系统评估 LLM 表格任务中的数据引用错误(DRE)，训练 4B 轻量 critic 模型检测 DRE（F1 78.2%），通过 critic 过滤将答案准确率最高提升 12% | [When LLMs Read Tables Carelessly: Measuring and Reducing Data Referencing Errors](https://arxiv.org/abs/2606.32029) |
| Agent 框架 | 将 LLM Agent 技能组合形式化为结构化序列预测，提出 SkillComposer 用约束自回归解码器联合预测技能子集/数量/顺序，在 SkillsBench 上为 GPT-5.2-Codex 提升 +23.1pp | [Generative Skill Composition for LLM Agents](https://arxiv.org/abs/2606.32025) |
| Agent 框架 | 将人类浏览器交互轨迹蒸馏为可读、可检索、可复用的自然语言技能并组织成技能图，论证浏览器 Agent 的可扩展性来自用户已表达的集体技能 | [Scalable Behaviour Cloning on Browser Using via Skill Distillation](https://arxiv.org/abs/2606.32014) |
| 代码生成 | 提出验证器引导的修复框架 AxDafny，迭代生成 Dafny 代码与证明，并引入 LCB-Pro-Dafny 基准，在 DafnyBench 上验证成功率达 92.7% | [AxDafny: Agentic Verified Code Generation in Dafny](https://arxiv.org/abs/2606.32007) |
| 工具调用/安全 | 神经符号框架，将组织政策转为可执行逻辑规则，LLM 回答局部抽取问题、符号评估器判定合规，使文档审查可检查、可测试 | [PolicyGuard: From Organizational Policies to Neuro-Symbolic Compliance Review Engines](https://arxiv.org/abs/2606.32004) |
| 训练方法/安全 | 揭示用自生成 QA 微调/蒸馏 LM 的隐藏脆弱性：生成器非均匀覆盖且易被指令式文本劫持，过滤指令段可将注入合规率从 88% 降至 13% | [Self-Study Reconsidered: The Hidden Fragility of Learning from Self-Generated QA](https://arxiv.org/abs/2606.32002) |
| Agent 评估/多模态 | 构建 72 场人机软件教学的多模态数据集，发现模型教练偏重直接指令、缺乏解释与视觉 grounding，学习者被动跟随而无深度参与 | [DigitalCoach: Communication and Grounding Gaps in Human and Agentic Computer Use Coaching](https://arxiv.org/abs/2606.31980) |
| Agent 评估/安全 | 提出 NCP-ToM，评估 Agent 通过行动（非对话）诱导他人信念状态的能力，GPT-5 在约 80% 任务成功，强调社交推理需 agentic 评估 | [Theory of Mind and Persuasion Beyond Conversation: Assessing the Capacity of LLMs to Induce Belief States via Planning and Action](https://arxiv.org/abs/2606.31916) |
| 安全/多模态 | 提出 MARS，无需多模态安全数据，直接从 LLM backbone 提取文本拒绝方向并校正模态错配，为多模态 LLM 注入安全性的训练免费方法 | [Harnessing Textual Refusal Directions for Multimodal Safety](https://arxiv.org/abs/2606.31876) |
| Agent 框架 | 端到端 agentic 框架：Co-Scientist Agent 将自然语言转为分析计划，Compute Agent 在 Frontier 超算上调度 ViT 分割，将分析从数周缩到秒级交互 | [An Agentic AI Framework to Accelerate Scientific Discovery in Plant Phenotyping](https://arxiv.org/abs/2606.31831) |
| LLM 推理/工具调用 | 将大规模车辆路径分解建模为迭代决策，用 LLM 作高层决策者分析状态并选择性应用聚类/平衡/细化算子，在 50 万客户规模上表现稳健 | [Adaptive Cluster-First Route-Second Decomposition for Industrial-Scale Vehicle Routing](https://arxiv.org/abs/2606.31820) |
| 模型压缩/量化 | 证明量化开源小模型在 16GB 显存本地运行可匹敌闭源 API 准确性，在 LM-数据库系统中实现 390 倍成本降低与 3.8 倍延迟降低 | [Large Databases Need Small, Open-Weight Language Models](https://arxiv.org/abs/2606.31808) |
| LLM 推理 | 提出 RAISE，将约束最坏实例搜索嵌入 LLM 驱动的进化启发式设计循环，使生成的启发式在分布偏移下保持鲁棒（现有方法最多退化 19 倍） | [RAISE: LLM-based Automated Heuristic Design with Robust Adversary Instance Search](https://arxiv.org/abs/2606.31801) |
| 模型压缩/训练方法 | 研究三种计算高效 LM 训练技术：选择性 token 监督、深度压缩+循环恢复（48 层压到 6 层）、压缩专家融合(MoEE)，在 1B 参数模型上验证 | [CHERRY: Compressed Hierarchical Experts with Recurrent Representational Yield](https://arxiv.org/abs/2606.31796) |
| 多 Agent | 自演化多 Agent 系统 ProtoPilot，含层级可验证性、多 Agent 编排与运行时技能库，实现协议转代码 89.5% 通过率并经湿实验验证 | [A Self-Evolving Agentic System for Automated Generation and Execution of Biological Protocols (ProtoPilot)](https://arxiv.org/abs/2606.31763) |
| 多模态/LLM 评估 | 评估 VLM 在协作对话中区分"可共享"与"已共享"信息的能力，发现 VLM 过度预测对齐，依赖静态参照线索而非追踪对话 grounding 历史 | [Seeing Is Not Sharing: Some Vision-Language Models Overestimate Common Ground in Asymmetric Dialogue](https://arxiv.org/abs/2606.31719) |
| Agent 框架/Agent 评估 | 全自动 AI 研究系统 FARS，自主完成构思/规划/实验/写作，首次公开部署产出 166 篇覆盖 67 个 AI/ML 主题论文并经 282 份结构化评审 | [FARS: A Fully Automated Research System Deployed at Scale](https://arxiv.org/abs/2606.31651) |
| RLHF/训练方法 | 提出相对惊奇指数 RSI 耦合 token 熵与选中概率，设计熵自适应过滤 RSI-S，在 AIME/AMC 上比 GRPO 提升 2–3 个点 | [Which Tokens Matter? Adaptive Token Selection for RLVR with the Relative Surprisal Index](https://arxiv.org/abs/2606.31575) |
