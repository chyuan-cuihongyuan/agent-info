# 2026-07-03 AI 资讯

| 研究方向 | 大概讲了什么 | 文章链接 |
|---------|------------|---------|
| 大模型 | 字节跳动 Seed2.0 模型卡，针对长尾知识与复杂指令遵循两大难题，构建面向真实需求的评估体系，在推理、视觉理解与搜索能力上达世界领先，初步具备处理复杂真实任务的能力 | [Seed2.0 Model Card: Towards Intelligence Frontier for Real-World Complexity](https://arxiv.org/abs/2607.00248) |

| 安全/对齐 | 扩展有限理性提出"有界道德"框架，沿道德广度与道德深度两个正交维度形式化道德问题的计算需求，指出道德对齐取决于道德推理能力的规模与分配而非直接模仿人类判断 | [Bounded Morality: Defining the Space of Moral Computation](https://arxiv.org/abs/2607.00002) |

| Agent 框架 | 提出 Agentic Transaction Processing (ATP)，将 Agent 生成动作视为不受信提案，经确定性准入与可执行约束集才提交，通过追加日志、依赖安全补偿证明四项安全性质并实现有界反应式修复 | [Mnemosyne: Agentic Transaction Processing for Validating and Repairing AI-generated Workflows](https://arxiv.org/abs/2607.00269) |
| Agent | 将页面级幻灯片个性化建模为逆向规划问题，通过对干净幻灯片故意腐蚀再做结构去噪作为可验证任务，双 Agent 经 RL 协同精炼可执行设计，证明结构去噪是 PSP 的一致代理且严格降低策略梯度方差 | [Personalization as Inverse Planning: Learning Latent Design Intents for Agentic Slide Generation via Structural Denoising](https://arxiv.org/abs/2607.00407) |
| 工具调用/Agent 评估 | 200 道多选题基准评估工具增强 Agent 在确定性水文地球化学模拟上的表现，发现工具访问提升整体准确率但存在非单调回归，倡导报告 item 级保留与输出访问敏感性而非仅看准确率 | [PHREEQC-MCQ-200: A Diagnostic Benchmark for Tool-Augmented Scientific Simulator Agents](https://arxiv.org/abs/2607.00436) |
| 多 Agent/LLM | 闭环框架 Agri-SAGE 将检索接地多 Agent LLM 推理与 APSIM 生物物理模拟结合生成并验证农艺建议，Tree of Thoughts 峰值产量最高，Reflexion 借跨季情景记忆以更低算力达可比效果 | [Agri-SAGE: Simulation-Grounded Multi-Agent LLM for Context-Aware Agricultural Advisory Generation](https://arxiv.org/abs/2607.00454) |
| 安全/对齐 | 发现越狱在 prompt 编码阶段抑制拒绝或有害方向即可成功；提出 HARC 微调将有害性与拒绝方向在 prompt 和响应位置耦合，干预限定于子空间不损通用能力，在六基线中达最佳鲁棒-能力-可用权衡 | [HARC: Coupling Harmfulness and Refusal Directions for Robust Safety Alignment](https://arxiv.org/abs/2607.00572) |

| Agent 框架/安全 | SEA 架构将自修改限制在小型 steering adapter 与冻结基座模型外层的版本化 harness，每次修改须经 anytime-valid 闸门出具可审计证书，在 SWE-bench Verified 上隔离出 +4/+5 贡献且防止回退 | [Self-Evolving Agents with Anytime-Valid Certificates](https://arxiv.org/abs/2607.00871) |
